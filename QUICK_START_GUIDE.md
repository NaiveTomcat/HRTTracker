# HRT Tracker 快速开始指南

## 功能概述

HRT Tracker 是一个用于跟踪和预测雌激素替代治疗(HRT)血药浓度的 Android 应用。

### 主要功能

1. **血药浓度可视化**
   - 实时显示雌二醇血药浓度曲线
   - 显示当前浓度和等级评估
   - 标记给药时间点
   - 可交互的图表界面

2. **用药记录管理**
   - 添加、编辑、删除用药记录
   - 支持多种给药途径：注射、口服、舌下、凝胶、贴片
   - 自动持久化保存
   - 修改后自动重新计算浓度曲线

3. **智能模拟计算**
   - 基于真实药代动力学模型
   - 自动选择最近30天或至少20次给药记录
   - 预测未来15天的浓度变化
   - 15分钟精度的时间步长

## 使用流程

### 1. 添加用药记录

1. 打开应用，点击底部导航栏的"记录"标签
2. 点击右下角的"+"按钮
3. 填写用药信息：
   - 选择给药途径（注射/口服/舌下/凝胶/贴片）
   - 选择药物类型（E2/EB/EV/EC/EN）
   - 输入剂量
   - 选择时间
   - 根据给药途径填写额外信息
4. 点击"保存"

### 2. 查看浓度曲线

1. 点击底部导航栏的"主页"标签
2. 查看当前浓度信息卡片
3. 查看血药浓度曲线图
4. 参考浓度等级说明

### 3. 编辑用药记录

1. 在"记录"页面，点击任意用药记录
2. 修改信息
3. 点击"保存"或"删除"
4. 应用自动重新计算浓度

## 浓度等级参考

| 等级 | 浓度范围 (pg/mL) | 说明 |
|------|-----------------|------|
| 偏低 | < 50 | 可能需要增加剂量 |
| 正常下限 | 50-100 | 接近正常范围 |
| 正常 | 100-200 | 理想范围 |
| 正常上限 | 200-300 | 接近上限 |
| 偏高 | 300-400 | 可能需要减少剂量 |
| 超高 | > 400 | 建议咨询医生 |

*注意：这些范围仅供参考，具体目标应咨询医生*

## 支持的给药途径

### 1. 肌肉注射 (INJECTION)
- 支持的酯类：EB, EV, EC, EN
- 需要输入剂量(mg)

### 2. 口服 (ORAL)
- 仅支持 E2
- 需要输入剂量(mg)

### 3. 舌下含服 (SUBLINGUAL)
- 支持 E2 和 EV
- 需要输入剂量(mg)
- 可选择吸收等级

### 4. 凝胶 (GEL)
- 仅支持 E2
- 需要输入剂量(mg)和涂抹面积(cm²)

### 5. 贴片 (PATCH_APPLY/PATCH_REMOVE)
- 仅支持 E2
- 两种模式：
  - 按剂量：输入总剂量(mg)和预计佩戴时间
  - 按释放速率：输入释放速率(µg/day)

## 技术说明

### 模拟算法

应用使用三室药代动力学模型：
1. 吸收室（对于非注射途径）
2. 酯类水解室（对于酯类药物）
3. 中心室（血浆）

模型参数基于文献数据，包括：
- 分布容积
- 清除速率
- 吸收速率
- 水解速率

### 数据存储

- 使用 Room 数据库本地存储
- 所有用药记录自动保存
- 支持数据持久化

### 隐私

- 所有数据仅存储在本地设备
- 不会上传到任何服务器
- 不需要网络连接

## 注意事项

1. **医学建议**: 本应用仅用于信息和教育目的，不能替代专业医疗建议
2. **数据准确性**: 预测基于理论模型，实际血药浓度可能因个体差异而有所不同
3. **定期检查**: 建议定期进行实验室血检以验证预测准确性
4. **剂量调整**: 任何剂量调整都应在医生指导下进行

## 常见问题

### Q: 为什么我的浓度曲线与实际血检结果不同？

A: 药代动力学模型基于平均参数，个体差异（代谢率、体重、肝功能等）会影响实际浓度。建议使用实际血检结果校准。

### Q: 我可以删除所有记录吗？

A: 可以，在记录列表中逐一删除，或者清除应用数据。

### Q: 如何备份我的数据？

A: 当前版本暂不支持导出功能，计划在未来版本中添加。

### Q: 应用支持哪些语言？

A: 当前仅支持中文，将来可能添加更多语言支持。
